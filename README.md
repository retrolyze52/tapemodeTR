# TypeRacer: Tape Mode

A UserScript that replaces the standard TypeRacer input box with a customizable, scrolling "tape" style display (similar to Monkeytype).

## Installation

1. Install a script manager extension like **Tampermonkey** or **Violentmonkey**.
2. [Click here to install the script](https://github.com/retrolyze52/tapemodeTR/raw/refs/heads/main/tapemode.user.js).

## Features

* **Visual Overhaul:** Smooth scrolling tape with customizable dimensions and opacity.
* **Theming:** Change fonts (Google Fonts included), colors, and cursor styles.
* **Pacemaker Mode:** Race against a "Ghost" cursor at a fixed WPM speed.
* **Live WPM:** See your estimated WPM in real-time above the caret.
* **Custom Layouts:**
    * **Cover Mode:** Fits perfectly over the existing game text.
    * **Manual Mode:** Drag and drop the tape anywhere on the screen.

## Usage

1. Go to `play.typeracer.com`.
2. Click the **Gear Icon (⚙️)** in the bottom right corner to open the settings menu.
3. Configure your fonts, colors, and ghost speed. Settings are saved automatically.
